# Gurobi

Gurobi 是目前应用最广泛的数学规划求解器之一，支持线性规划（LP）、整数规划（IP）、混合整数规划（MIP）、二次规划（QP）、二次约束规划（QCP）等模型。

本专题主要记录 Gurobi 在 Python 中的常用建模方法、API 使用以及项目实践。
