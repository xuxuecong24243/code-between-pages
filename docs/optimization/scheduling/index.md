# Scheduling

调度优化（Scheduling）是运筹优化（Operations Research）中的重要研究方向，广泛应用于制造、物流、云计算、医疗和生产计划等领域。

本栏目主要整理经典调度问题、数学模型、求解算法以及 Gurobi 实现，内容参考《Scheduling: Theory, Algorithms, and Systems》以及相关学术论文。
