# Routing

路径优化（Routing）是运筹优化（Operations Research）的重要研究方向，广泛应用于物流配送、城市交通、仓储运输、无人机配送以及供应链优化等领域。

本栏目主要整理经典路径优化问题、数学模型、求解算法以及 Gurobi 实现。
