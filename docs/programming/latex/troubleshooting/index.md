# 🛠️ LaTeX 故障排查

记录 LaTeX 使用过程中遇到的实际问题、排查过程及解决方案，主要涵盖编译错误、排版异常、宏包冲突、图片与字体问题等。

本模块以真实问题为主，不系统讲解 LaTeX 基础语法。

> 持续更新中……

---

## 📚 问题分类

### 编译与环境

- 编译失败
- 编译器选择错误
- TeX Live、MiKTeX 环境异常
- 宏包缺失或版本冲突
- Overleaf 与本地编译结果不一致

### 图片与 PDF

- 图片无法显示
- 图片位置或尺寸异常
- 图片颜色变化
- RGB 与 CMYK 色彩问题
- EPS、PDF、PNG、JPG 格式兼容问题
- PDF 输出异常

### 字体与编码

- 中文乱码
- 字体缺失
- 字体无法加载
- XeLaTeX、LuaLaTeX 与 pdfLaTeX 兼容问题

### 参考文献

- BibTeX 或 Biber 编译失败
- 参考文献不显示
- 引用显示为问号
- 文献样式异常
- `.bib` 文件格式错误

### 排版与宏包

- 宏包冲突
- 数学公式报错
- 表格超出页面
- 浮动体位置异常
- 页眉、页脚、目录和编号异常
- 模板样式与自定义设置冲突

## 🔍 排查原则

处理 LaTeX 问题时，通常按照以下顺序排查：

1. 查看完整报错信息和 `.log` 文件。
2. 确认使用的编译器与编译流程。
3. 检查导言区中的宏包和相关配置。
4. 检查图片、字体、参考文献等外部文件。
5. 尝试构造最小可复现示例。
6. 逐步注释代码，定位冲突位置。
7. 清理辅助文件后重新编译。
8. 验证修改是否引入新的排版问题。
